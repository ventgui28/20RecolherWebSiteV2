# Guia de Contribuição

Obrigado pelo seu interesse em contribuir para o projeto **20recolher**! 🌍♻️

## 🌿 Como Contribuir

1. Faça um **Fork** do projeto.
2. Crie uma **Branch** para a sua funcionalidade (`git checkout -b feature/minha-melhoria`).
3. Faça os seus **Commits** seguindo o padrão [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/):
   - `feat:` para novas funcionalidades.
   - `fix:` para correção de bugs.
   - `docs:` para alterações na documentação.
   - `style:` para alterações visuais que não afetam a lógica.
   - `refactor:` para alterações no código que não corrigem bugs nem adicionam funcionalidades.
4. Faça o **Push** para a sua Branch (`git push origin feature/minha-melhoria`).
5. Abra um **Pull Request** para a branch `develop`.

## 🛠️ Regras do Projeto

- **Tecnologias:** Use Next.js 15 e Tailwind CSS v4.
- **Componentização:** Siga a estrutura de componentes atómicos e modulares.
- **Dados:** Não escreva textos diretamente nas páginas. Use o ficheiro de constantes adequado em `src/constants`.
- **Estilo:** Mantenha a consistência visual com a paleta de verdes da marca.
